1.-Para poder ejecutar el proyecto es necesario instalar node
2.-Descargar el proyecto (Solo se contiene carpeta SRC)
3.-Crear un nuevo proyecto en react (npx create-react-app nameTest)
4.-Sustituir la carpeta src que se encuentra dentro del proyecto creado anteriormente
5.-Desde CMD o Terminal iniciar el proyecto desde la direccion seleccionada con npm start

El aplicativo no se conecta a ninguna BD, solo esta en trabajndo con los props del mismo
