1.-Para poder ejecutar el proyecto es necesario instalar nodeJS
2.-Descargar el proyecto (carpeta SRC) ya que en git no deja subir mas mb de memoria porque es cuenta free
3.-Crear un nuevo proyecto en react (npx create-react-app "nombreProyectoQueSea")
4.-Sustituir la carpeta src que se encuentra dentro del proyecto creado anteriormente
5.-Desde CMD o Terminal iniciar el proyecto desde la direccion seleccionada con npm start

El aplicativo no se conecta a ninguna BD, solo esta en trabajndo con los props del mismo para poder ejecutar sin problema
